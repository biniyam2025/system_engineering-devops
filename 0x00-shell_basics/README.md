# 0x00. Shell, basics

## Description
This project focuses on learning the basics of shell commands and navigation in a Unix/Linux environment. It covers fundamental concepts of working with the command line interface.

## Learning Objectives
At the end of this project, you should be able to explain:

### General
- What is the shell
- What is the difference between a terminal and a shell
- What is the shell prompt
- How to use the history (the basics)

### Navigation
- What do the commands or built-ins `cd`, `pwd`, `ls` do
- How to navigate the filesystem
- What are the `.` and `..` directories
- What is the working directory, how to print it and how to change it
- What is the root directory
- What is the home directory, and how to go there
- What is the difference between the root directory and the home directory of the user root
- What are the characteristics of hidden files and how to list them
- What does the command `cd -` do

### Looking Around
- What do the commands `ls`, `less`, `file` do
- How do you use options and arguments with commands
- Understand the ls long format and how to display it

### Manipulating Files
- What do the commands `cp`, `mv`, `rm`, `mkdir` do
- What are wildcards and how do they work
- How to use wildcards

### Working with Commands
- What do `type`, `which`, `help`, `man` commands do
- How to read a man page
- What are man page sections
- What are the section numbers for User commands, System calls and Library functions

### Keyboard Shortcuts
- Common shortcuts for Bash
- What does `Ctrl+C`, `Ctrl+D` do
- What does `Tab` completion do

## Requirements
- Allowed editors: `vi`, `vim`, `emacs`
- All scripts tested on Ubuntu 20.04 LTS
- All scripts should be exactly two lines long
- All files should end with a new line
- The first line of all your files should be exactly `#!/bin/bash`
- All files must be executable
- You are not allowed to use backticks, `&&`, `||` or `;`

## Tasks
This project contains scripts that demonstrate basic shell commands and operations.
