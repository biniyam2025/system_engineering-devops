# 0x05. Processes and Signals

This project contains Bash scripts focused on understanding Linux processes and signals.

## Project Description

This project explores process management in Linux, including:
- Process identification (PIDs)
- Listing and managing processes
- Understanding and handling signals
- Process control and termination

## Learning Objectives

By completing this project, you will understand:
- What is a PID
- What is a process
- How to find a process' PID
- How to kill a process
- What is a signal
- What are the 2 signals that cannot be ignored

## Tasks

### 0. What is my PID
Bash script that displays its own PID.

**File:** `0-what-is-my-pid`

## Requirements

- All files are interpreted on Ubuntu 20.04 LTS
- All Bash scripts are executable
- All Bash scripts pass Shellcheck (version 0.7.0) without any error
- The first line of all Bash scripts is `#!/usr/bin/env bash`
- The second line of all Bash scripts is a comment explaining what the script does
