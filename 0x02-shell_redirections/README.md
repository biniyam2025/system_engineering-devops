# 0x02. Shell, I/O Redirections and filters

This project contains shell scripts demonstrating I/O redirections and filters.

## Learning Objectives

- Understanding shell I/O redirections
- Working with standard input, output, and error
- Using filters and text processing commands
- Combining commands and filters

## Requirements

- All scripts are tested on Ubuntu 20.04 LTS
- All scripts start with `#!/bin/bash`
- All files must be executable

## Tasks

### 0. Hello World
Write a script that prints "Hello, World", followed by a new line to the standard output.
